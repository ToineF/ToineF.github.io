Game Jams - Solo

A list of short projects I made by myself, making both the programming, the art and the design.

  • This is a Tomato

  • Itch.io

Project : You got a new job at a fastfood restaurant where you can't make a single mistake or you'll be fired!
But this workplace is a bit peculiar. Sometimes a steak is a tomato. Or maybe a lettuce.
Context : Brain Teaser made for the Brackeys Game Jam 2025.1 in 72h using Unity.
Results : 20th out of 2200 games overall. 19th in enjoyment.
My role : I did everything (Concept, Programming, 2D Art, Pacing...) except for the music and sound effects.




  • Mrs. Rose's Head Garden

  • Itch.io

Project : As Mrs. Rose's gardener, she demands you to plant some heads in her beautiful garden.
Context : Sokoban made for the GMTK Game Jam 2023 in 48h using Game Maker.
Results : 89th out of 6700 games overall. 39th in enjoyment and 54th in presentation.
My role : I did everything (Concept, Programming, 2D Art, Sound effects, Level Design...) except for the music.




  • Express Delivery

  • Itch.io

Project : The quick postman is having a nice day at work. His job: deliver a few packages by running REALLY fast!
Context : 3D Speedrun-oriented Runner made for the Speedjam 4 in 72h using Unity.
My role : I did everything (Concept, Programming, 2D/3D Art, VFX, Dialogs...) except for the music and sound effects.




  • Tako, Defender of the Earth

  • Itch.io

Project : Tako is an Octopus Sausage that escaped Earth's orbit, and it has a mission: Defend the Earth.
Context : Arcade game made as a personal project to learn Godot.
My role : I did everything (Concept, Programming, 2D Art, Pacing...) except for the music and sound effects.
